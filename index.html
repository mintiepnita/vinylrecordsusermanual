🛈 About This Website

📢📢📢 This website serves as a user manual for vinyl records, designed as part of a web-to-print class project. The starting point was to use selected objects, in this case, vinyl records, and create a user manual out of it. The project required using just one Google tool as a print design tool, so Google Docs was chosen to design ASCII art for the manual.📜 🪄 📖 ✨

✎ᝰ。With this vinyl records manual website, you can customize their own ASCII art pages for a vinyl records manual book. Select different options to personalize your manual, and then print it out for physical use. 🎨 🖌 🖨️
